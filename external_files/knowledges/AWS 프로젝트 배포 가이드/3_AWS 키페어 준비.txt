[AWS 키페어 준비]
- 키페어는 AWS 의 EC2 인스턴스에 접속하기 위한 비밀번호를 저장한 파일과 같은 것입니다.
    이것이 없다면 EC2 로 만든 인스턴스에 접속을 할 수 없습니다.
    앞으로 이것을 사용하여 EC2 에 접속해서 조작할 것입니다.

- 키페어 생성
    1. AWS 콘솔의 EC2 페이지로 이동합니다.
    2. 네트워크 및 보안 탭의 키페어 페이지로 이동합니다.
    3. 키페어 생성 버튼을 누릅니다.
    4. 키페어 설정을 합니다.
        이름에는 임의의 이름을 넣고, 키페어 유형은 RSA,
        프라이빗 키 파일 형식은, 저의 경우 OpenSSH 를 사용할 것이므로 .pem 형식으로 한 후,
        키 페어 생성 버튼을 누르면 생성된 키페어가 다운로드 됩니다.

- 키페어 파일 보관
    외부에는 공개되서는 안될 중요한 파일이므로 잘 저장해둡시다.
    저의 경우는 ServerInfo 라는 폴더를 만들고,
    그 안에 Files 디렉토리를 만든 후 여기에 파일을 넣어둡니다.
